# Конвертер Jira CSV → XLSX

Десктопное приложение для Windows, которое преобразует экспорт задач Jira (CSV) в форматированные Excel-таблицы (XLSX) — и обратно.

**Автор:** [maxsteff](https://t.me/maxsteff)

---

## Возможности

- **Drag & Drop** — перетащите CSV прямо в окно (или выберите через «Обзор»)
- **Склейка нескольких CSV** в один XLSX с автоматическим удалением дублей
- **Очистка заголовков** — `Пользовательское поле (Статус)` → `Статус`
- **Фильтрация и порядок колонок** через файл `columns.txt`
- **Произвольные префиксы обрезки** через секцию `[strip]` в `columns.txt`
- **Конвертация дат** Jira (`05/фев/26 12:00 AM`) → `05.02.2026`
- **Автоопределение кодировки** CSV — UTF-8 (BOM/без), CP1251, Latin-1
- **Логирование** в `app.log` рядом с `.exe` — для разбора проблем
- **22 стиля таблиц Excel** с визуальным превью (светлые / средние / тёмные)
- **Обратная конвертация** XLSX → CSV для загрузки обратно в Jira
- **Тёмная и светлая тема** — автоопределение из реестра Windows
- **8 языков интерфейса** — RU, EN, DE, FR, ES, ZH, AR, PT

## Установка и запуск

### Готовый .exe

Скачайте `Конвертер_Jira_CSV_XLSX.exe` из [Releases](../../releases) и запустите — установка не нужна.

### Из исходного кода

```bash
git clone https://github.com/<user>/jira-csv-xlsx-converter.git
cd jira-csv-xlsx-converter

pip install -r requirements.txt
python app.py
```

### Сборка .exe самостоятельно

```bash
pip install pyinstaller
python -m PyInstaller app.spec
# → dist/Конвертер_Jira_CSV_XLSX.exe
```

## Файл columns.txt

Положите `columns.txt` рядом с `.exe`. Программа подхватит его автоматически.

```
# Строки с # — комментарии

[columns]
Ключ проблемы
Тема
Статус
Дата регистрации замечания
Плановая дата устранения

[strip]
Пользовательское поле 
Custom field 
```

- **`[columns]`** — список колонок для сохранения (порядок сохраняется в XLSX)
- **`[strip]`** — префиксы для автоочистки имён; пробел в конце строки значим

Нет файла — кнопка **«Создать пример»** сгенерирует шаблон на основе текущего CSV.
